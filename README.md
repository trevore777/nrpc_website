# NRPC Site Starter

A lightweight Next.js rebuild for Northern Rivers Pentecostal Church.

## Why this build

This starter is focused on first-time visitors:
- clearer homepage
- dedicated Plan Your Visit page
- simpler beliefs page
- better mobile hierarchy
- SEO-ready metadata and Church schema
- AI-ready content structure

## Local run

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push the folder to GitHub.
2. Import the repo into Vercel.
3. Deploy with default settings.

## Recommended next additions

- Real photos of the congregation and venue
- Google Maps embed
- Live contact handling via Formspree, Resend or Gmail API
- Sermon video/audio embeds
- AI assistant using an OpenAI-powered FAQ route
- Basic CMS using Markdown, Notion or Sanity

## Suggested cost profile

- Vercel hobby hosting: free for a small church brochure site
- Domain renewal: typically low annual cost
- Form handling: free or low-cost depending on provider
- AI assistant: usage-based and can be kept very small by limiting it to FAQs
